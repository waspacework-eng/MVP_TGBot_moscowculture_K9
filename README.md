## Telegram-бот для спонтанных культурных походов в Москве

Описание:

Проект предполагает разработку Telegram-бота для интеллектуального анализа и структурирования культурных мероприятий. 

Система обрабатывает текстовые анонсы событий (концерты, выставки, лекции, спектакли и другие форматы досуга) с использованием больших языковых моделей и автоматически извлекает ключевые параметры мероприятия. Бот выделяет название события, категорию, дату и время проведения, место, стоимость участия, формат мероприятия и тематические характеристики, после чего преобразует неструктурированный текст в стандартизированную карточку события. 

На основе извлечённых данных формируется краткое аналитическое резюме мероприятия, определяется его тип (например, камерное, массовое, образовательное, семейное), а также выделяются тематические теги и потенциальная целевая аудитория. Информация сохраняется в базе данных и используется для последующего поиска и фильтрации.
